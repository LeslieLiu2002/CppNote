# 什么是枚举
- 本质上是一组命名的值
- 用于给整数赋予有意义的名字
- 本质是整数
___
# 基本用法
```c++
enum Level
{
    Error,Warning,Info // 注意末尾不要加分号
};// 这里要加分号

Level l1 = 0; // 不能将int右值赋给Level这个enum变量，哪怕Level给0赋予了意义：Error=0
Level l2 = Error; // 只能将enum里定义的值(枚举值)赋给Level变量，枚举值可以直接使用
```
- 定义类型：可以把`enum`用作一种数据类型，==限制变量只能被赋予该枚举内定义好的特定值==，而不是随意数字
- 赋值：可通过手动指定枚举元素对应的值，紧随其后的元素就是该值的累加值，默认第一个元素的值是0
- 制定底层数据类型：默认情况，枚举是32位的整数(int)，可以通过冒号`:`指定其底层数据类型，`enum Example : unsigned char`这样就只占8位
> [!tip] 命名冲突
> 普通`enum`不是一个单独的命名空间。如果在一个命名空间内定义了`enum`中的`Error`，同时定义了一个名为`Error()`的函数，那就会导致命名冲突。解决方法是在枚举值前面加上前缀(一般是枚举名，例如"LevelError")，或者用`enum class`(强类型枚举)
