# 什么是指针
> [!tip] 指针就是一个整数用来表示内存的地址。

它不受所指数据的类型而改变，无论指向什么数据，它都是整数

0不是一个有效的地址，我们不能从地址为0的内存中读取和写入内容；但指针指向0是有效状态，即指针无效/空指针

因为指针和指向数据的类型无关，所以下面的代码有效：
```c++
int var = 8;
void* ptr = &var;
```
这里的`void*`被称为万能/通用指针，用来存储数据的起始地址。但它不能读取内容(但可读取地址)或者解引用(写入)，因为它不知道存储的数据占多少字节。想要能解引用可以用下面的代码：
```c++
int var = 8;
int* ptr = &var; 
*ptr = 10; //解引用
```
此时的`ptr`知道内容占位大小，如果将指针设置为`double*`需要做强制类型转换：
```c++
double* ptr = (double*)&var;
```
因为`&var`是一个`int*`类型的右值，同时`int`和`double`的占位数量不一样，因此要转换。否则在解引用时会错误读取，即超过`int`的四个字节多读4个字节变成`double`
